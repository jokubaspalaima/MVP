Capacitor (iOS native app — needs Mac):

npm install @capacitor/core @capacitor/cli @capacitor/ios
npx cap init not2day app.not2day.app
npx cap add ios
npx cap copy
npx cap open ios
Build + archive in Xcode
Submit via App Store Connect
